# Color System

This design system uses an advanced OKLCH-based color theming system that automatically generates:
- **19 perceptually uniform color scales** (50-950) from any base color
- **Hundreds of intelligent light/dark pairings** that switch automatically
- **Zero-config theme switching** via `color-scheme` property

## How It Works

Set a single base color:
```css
--theme-color-primary: red;
```

Get a complete color ecosystem with automatic light/dark variants using the `light-dark()` CSS function.

## Usage Examples

```css
/* Background that adapts to mode */
.card {
  background-color: var(--color-neutral-50-900);
  border: 1px solid var(--color-neutral-200-700);
  color: var(--color-neutral-900-100);
}

/* Primary button with automatic contrast */
.btn-primary {
  background-color: var(--color-primary-600-400);
  color: var(--color-primary-50-950);
}

.btn-primary:hover {
  background-color: var(--color-primary-700-300);
}
```

## Theme Customization

Override base colors in your CSS:

```css
:root {
  --theme-color-primary: oklch(0.5 0.2 200); /* Custom blue */
  --theme-color-secondary: #ff6b35;          /* Orange */
  --theme-color-accent: var(--color-green-500); /* Predefined green */
}
```

The system will automatically generate complete color scales and light/dark pairings from these base colors.
